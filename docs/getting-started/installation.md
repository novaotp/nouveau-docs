# Installation

Requirements :

- make
- g++

## Getting Started

1. Clone the repository

   ```bash
   git clone https://github.com/novaotp/nouveau
   ```

1. Build the compiler

   ```bash
   make -s build
   ```

1. Execute a file

   ```bash
   ./bin/nouveau.exe ./path/to/your/file.nv
   ```

## Testing

1. Run the test suite

   ```bash
   make -s test
   ```
